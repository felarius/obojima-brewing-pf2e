Obojima Brewing for Pathfinder 2e
A Foundry VTT module for Pathfinder 2e that brings an expanded ingredient-based brewing and monster
harvesting system inspired by Obojima.

Failed brews create mysterious unidentified consumables.
Critical failures create unstable magical brews that appear normal until consumed, at which point one of
many hidden magical catastrophes is unleashed.
